# Proyecto de Programación - 2ª Evaluación 🚀

Este proyecto ha sido desarrollado como parte del **Grado Superior en Desarrollo de Aplicaciones Web (DAW)**. Se centra en la lógica de programación avanzada y la gestión eficiente de datos, demostrando la implementación de patrones de diseño y estructuras de datos complejas en PHP.

El objetivo principal fue crear un sistema escalable mediante el uso de **Interfaces y Repositorios**, garantizando la separación de responsabilidades y facilitando el mantenimiento del código profesional.

## 🛠️ Características técnicas

* **Interacción con Arrays:** Gestión dinámica y eficiente de datos estructurados.
* **Interfaces:** Implementación de contratos para asegurar un código desacoplado y organizado.
* **Patrón Repository (REPOs):** Abstracción de la lógica de acceso a datos para mejorar la testabilidad y el orden.
* **Lenguaje:** PHP 8.x con buenas prácticas.
* **Estilos:** CSS3 personalizado para una interfaz funcional.

## 💻 Instalación y ejecución

1.  **Clonar** el repositorio en tu máquina local.
2.  Asegurarte de tener un entorno de servidor local activo (como **XAMPP, WAMP o Laragon**) con soporte para PHP 8.
3.  Mover la carpeta del proyecto al directorio `htdocs` o `www`.
4.  Abrir `index.php` (o el punto de entrada principal) en tu navegador a través de `localhost`.
